# Executive Summary Format

Professional markdown format ideal for business stakeholders and quick overviews.

## Structure

```markdown
# {Title}

**Date:** {YYYY-MM-DD}
**Source:** {source reference}
**Type:** Executive Summary

---

## Key Takeaways

- {Key point 1}
- {Key point 2}
- {Key point 3}

## Summary

{2-3 paragraph summary of the main content}

## Notable Points

> "{Important quote or statement}"

---

*Generated from: {original source}*
```

## Best Used For

- Business presentations
- Stakeholder briefings
- Quick decision-making references
- Email summaries

## File Extension

`.md` (Markdown)

## Filename Pattern

`{source}_executive-summary_{YYYYMMDD}.md`
