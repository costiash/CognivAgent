# Key Points Format

Action-oriented format for quick reference and task tracking.

## Structure

```markdown
# Key Points: {Title}

**Date:** {YYYY-MM-DD}
**Source:** {source reference}

---

## Main Points

- {Main point 1}
- {Main point 2}
- {Main point 3}
- {Main point 4}
- {Main point 5}

## Action Items

- [ ] {Action 1}
- [ ] {Action 2}
- [ ] {Action 3}

## Important Quotes

> "{Key quote 1}"

> "{Key quote 2}"

## Quick Reference

| Topic | Summary |
|-------|---------|
| {Topic 1} | {One-line summary} |
| {Topic 2} | {One-line summary} |
| {Topic 3} | {One-line summary} |

---

*Extracted from: {original source}*
```

## Best Used For

- Quick briefings
- Task lists
- Study guides
- Review checklists

## File Extension

`.md` (Markdown)

## Filename Pattern

`{source}_key-points_{YYYYMMDD}.md`
