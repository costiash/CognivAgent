# Detailed Notes Format

Comprehensive documentation format for research and reference.

## Structure

```markdown
# {Title}

| Field | Value |
|-------|-------|
| Date | {YYYY-MM-DD} |
| Source | {source reference} |
| Duration | {if applicable} |
| Type | Detailed Notes |

---

{Brief introduction to the content and its context}

### Topic 3: {Title}

## Key Details

- **Detail 3:** {explanation}

## Quotes & References

> "{Quote 1}" — {attribution if known}

> "{Quote 2}" — {attribution if known}

## Action Items

- [ ] {Action item 1}
- [ ] {Action item 2}
- [ ] {Action item 3}

---

*Notes generated from: {original source}*
*Generated on: {date}*
```

## Best Used For

- Research documentation
- Meeting notes
- Training materials
- Reference guides

## File Extension

`.md` (Markdown)

## Filename Pattern

`{source}_notes_{YYYYMMDD}.md`
