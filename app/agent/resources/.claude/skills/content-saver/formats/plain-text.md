# Plain Text Format

Simple, universal format for maximum compatibility.

## Structure

```text
================================================================================
{TITLE}
================================================================================

Date: {YYYY-MM-DD}
Source: {source reference}
Type: {content type}

--------------------------------------------------------------------------------

{Main content body}

{Paragraphs separated by blank lines}

{Additional sections as needed}

--------------------------------------------------------------------------------

KEY POINTS:

* {Point 1}
* {Point 2}
* {Point 3}

--------------------------------------------------------------------------------

Source: {original source}
Generated: {date}

================================================================================
```

## Best Used For

- Universal sharing
- Email content
- Legacy systems
- Terminal/CLI viewing
- Maximum compatibility

## File Extension

`.txt`

## Filename Pattern

`{source}_{content-type}_{YYYYMMDD}.txt`

## Notes

- Use ASCII characters only (no special Unicode)
- Fixed-width dividers for visual separation
- Simple bullet points with asterisks
- No markdown formatting
